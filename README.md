# portfolio-app

Portfolio app built with Vue 3 + Vite + TypeScript.

## Prerequisites

- Node.js: 20.19+ or 22.12+
- npm (bundled with Node.js)

## Install

```bash
npm install
```

## Development

```bash
npm run dev
```

## Type check

```bash
npm run type-check
```

## Build

```bash
npm run build
```

## Preview production build

```bash
npm run preview
```

## Deploy to GitHub Pages

```bash
npm run deploy
```

## Notes

- The Vite base path is set to `/Portfolio-BUT2/` for GitHub Pages.
